# AI Study Assistant 🎓

An AI-powered tool built with **Python**, **Streamlit**, and **Google Gemini 1.5 Flash**. Upload any educational PDF or a photo of a textbook page to automatically generate:

- **Lecture Plans**: Structured 5-day teaching schedules.
- **Mock Tests**: 10-question quizzes with answer keys.
- **Study Notes**: Concise, bulleted summaries of key concepts.

## 🚀 How to Run Locally
1. Clone this repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
4. Enter your Gemini API Key in the sidebar.

## 🛠️ Built With
- [Streamlit](https://streamlit.io/) - UI Framework
- [Google Gemini API](https://aistudio.google.com/) - LLM & Vision
- [PyMuPDF](https://pymupdf.readthedocs.io/) - PDF Processing
